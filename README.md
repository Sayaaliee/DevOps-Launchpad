# 🚀 DevOps Launchpad

A personal DevOps portfolio web application showcasing skills, build versioning, and deployment capability.

---

## 👩‍💻 About the Project

DevOps Launchpad is a lightweight web application built to demonstrate:

- Basic frontend development (HTML/CSS)
- Docker containerization
- CI/CD pipeline integration
- Deployment on AWS EC2
- Version-controlled development using Git

This project is designed to simulate a real-world DevOps deployment workflow.

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Docker
- Git & GitHub
- Jenkins (Planned CI/CD)
- AWS EC2 (Deployment)

---

## 📦 Features

- Clean responsive UI
- Build version display
- Easy container deployment
- CI/CD ready structure
